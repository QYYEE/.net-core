- enum 的用法（enums文件夹）
  - enums枚举是值类型，数据直接存储在栈中，而不是使用引用和真实数据的隔离方式来存储。
  - 枚举（enum type）通常用来表示一组常量。由于枚举是强类型的，这在编程中给我们提供了极大的方便。
     例如：public enum Sex
        {
            男 = 0,
            女 = 1
        }
  如果只给男赋值，那么女=1；
